**What (สร้างอะไร):**
ช่วยออกแบบและสร้างโครงสร้างเว็บไซต์ Portfolio ส่วนตัว (Personal Portfolio Website) สำหรับนักเรียนชั้น ม.3 เพื่อใช้เป็นแฟ้มสะสมผลงานในการยื่นสัมภาษณ์เข้าศึกษาต่อชั้น ม.4

**Who (ใครใช้):**
กลุ่มผู้ใช้งานหลักคือ คณะกรรมการรับนักเรียนและคุณครูผู้สัมภาษณ์ (ระดับความรู้: เป็นผู้ใช้งานเว็บไซต์ทั่วไป ต้องการแพลตฟอร์มที่ใช้งานง่าย กดดูข้อมูลได้รวดเร็ว ไม่ซับซ้อน)

**Why (ทำไมถึงสร้าง):**
เพื่อนำเสนอประวัติส่วนตัว ทักษะ ผลงาน และกิจกรรมที่โดดเด่นอย่างเป็นระบบ ช่วยให้คณะกรรมการเห็นศักยภาพและความตั้งใจจริงผ่านรูปแบบออนไลน์ที่เข้าถึงง่าย และสร้างความประทับใจที่แตกต่างจากแฟ้มกระดาษทั่วไป

**Features (ทำอะไรได้บ้าง):**
ขอให้มีฟีเจอร์หลัก 4 หน้า/ส่วน ดังนี้:
1. **หน้า Home:** แนะนำตัวสั้นๆ (ชื่อ-สกุล, โรงเรียนเดิม, สายการเรียนเป้าหมายที่อยากเข้า) พร้อมพื้นที่ใส่รูปถ่ายที่ดูสุภาพ
2. **หน้า Profile:** สรุปประวัติส่วนตัว, ข้อมูลติดต่อ, และประวัติการศึกษา (เกรดเฉลี่ย)
3. **หน้า Activities & Awards:** พื้นที่แสดงผลงานและเกียรติบัตร โดยแบ่งหมวดหมู่ให้ชัดเจน (เช่น ด้านวิชาการ, ด้านกีฬา/ศิลปะ, ด้านจิตอาสาและกิจกรรมโรงเรียน)
4. **ปุ่ม Download:** มีปุ่ม Call-to-action ชัดเจน เพื่อให้กรรมการกดดาวน์โหลด Portfolio ตัวเต็มในรูปแบบไฟล์ PDF ได้

**Look (หน้าตาเป็นยังไง):**
เน้นสไตล์ Minimalist สะอาดตา เรียบง่ายแต่ดูเป็นระเบียบ ใช้โทนสีที่ดูสุภาพและเป็นทางการ (เช่น โทนสีขาว-น้ำเงิน, ขาว-เทา หรือสีประจำโรงเรียนเป้าหมาย) Layout ต้องเป็นแบบ Responsive Design คือสามารถเปิดดูบนหน้าจอมือถือและแท็บเล็ตของกรรมการได้สวยงาม ตัวหนังสือต้องใหญ่และอ่านง่าย

**How (Implement อย่างไร):**
สร้างโครงสร้างด้วย HTML, CSS และ JavaScript พื้นฐาน (หรือให้คำแนะนำเพื่อนำไปสร้างต่อบน Website Builder อย่าง Google Sites หรือ Canva)
*สิ่งที่ไม่อยากให้มี:* ไม่เอาเอฟเฟกต์ (Animation) ที่หวือหวาเกินไปจนทำให้เว็บไซต์โหลดช้า ไม่เอาฟอนต์ที่อ่านยาก และไม่เอาโครงสร้างเมนูที่ซับซ้อนต้องกดหลายคลิก
